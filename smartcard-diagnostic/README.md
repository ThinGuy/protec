# Smart Card Diagnostic Tool

Professional diagnostic application for government and enterprise smart card deployments.

## Supported Card Types

- PIV (Personal Identity Verification) cards
- PIV-I (Interoperable) cards
- Government-issued smart cards
- Enterprise PKI cards

## Features

- Real-time reader/card detection
- Comprehensive test suite
- FIPS 140-2/140-3 compliance validation
- Certificate monitoring
- Support bundle generation
- Ubuntu Desktop and Core support

## Installation

```bash
sudo snap install smartcard-diagnostic
```

## Usage

```bash
smartcard-diagnostic
```

## Privacy

No telemetry. Support bundles require user approval.
No PINs, passwords, or private keys collected.
